Local Trader

Local Trader is an Android application for LocalBitcoins. Use Local Trader to create and update advertisements, manage trades, download attachments, release and fund trades, contact advertisers, or to handle payments.

TRADE

With Local Trader you can search for buyers or sellers, release and fund trades, receive trade messages or create and manage advertisements.

WALLET

Local Trader offers an optional feature to send money from your LocalBitcoins wallet. You can scan or generate QR codes for payments or review wallet history.

SECURITY

Local Trader offers secure authentication through the LocalBitcoins webste. You can also switch the application to use one of LocalBitcoin's mirror sites.

Fully Featured

Local Trader uses the LocalBitcoins service and authentication to bring you a safe and secure way to fully mange your trades and wallet.

Using Local Trader with a LocalBitcoins Account

LocalBitcoins Account

A LocalBitcoins account is required to use Local trader. To get an account, please register for at LocalBitcoins.com or use the new mirror site at LocalBitcions.net for regions that may be blocked.

Security

Local Trader authenticates through the LocalBitcoins website using Oauth 2. Upon successful authentication you will be returned to the application. In order to send funds, you will also need a personal PIN code. View or Manage your PIN code on your Profile page.

Additional Information

Local Trader is not owned or operated by LocalBitcoins. Please contact LocalBitcoins Support directly for any account issues or questions. If you would like support for the Local Trader application, please visit our new Reddit Community page.
